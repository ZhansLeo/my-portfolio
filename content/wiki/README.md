# Wiki 记忆规则

1. **写入前先搜索** — 新条目写入前，必须在已有页面中搜索同义或高度重叠的主题，避免重复。
2. **优先更新已有页面** — 发现同义页面时，在现有页面上追加、修订，不创建新文件。
3. **区分来源** — 每页内容必须明确标记出处：
   - `## 我的原话` — 用户本人确认的观点、经历、反思；
   - `## Agent 总结` — Agent 对用户原话的整理、归纳或补充；
   - `## 外部来源` — 引用的第三方资料、论文、链接。
4. **不推测** — Agent 未经确认的推断、假设、延伸结论，不得写入"我的原话"部分。
5. **元数据要求** — 每页 frontmatter 必须包含 `updated`（更新日期）和 `tags`（标签数组）。
6. **页面互联** — 新页面必须至少包含一个指向已有 Wiki 页面的链接（形式：`[页面名](slug)`）。
7. **变更可追溯** — 任何对 Wiki 源文件的修改，必须经过 `git diff` 审查后才能提交。
